# docker.shell
Shell script to maintain docker containers

## bin/cmd.sh
up, down, pull container
	
## bin/up container -d
```
up mosca -d
```

## bin/down container
```
down mosca
```

## bin/pull container
```
pull mosca
```

## bin/clean

## bin/docker-compose
start container to run docker compose ui
```
bin/docker-compose -f compose/composeui/docker-compose.yml up
```
